# Nym平台

Nym是一个隐私平台,它提供了强大的网络级隐私，以对抗复杂的端到端攻击者，并使用盲目的、可重新随机化的、分散的证书进行匿名访问控制。

我们的目标是让开发者建立新的应用程序，或升级现有的应用程序，并具有其他系统所没有的隐私功能。
